## Khởi chạy Lab
Để chạy được bài lab, cần có sẵn Python. 

*   **Bước 1:** Cài đặt thư viện Flask. Mở terminal (CMD/PowerShell) tại thư mục chứa code và chạy lệnh:
    ```bash
    pip install flask
    ```
*   **Bước 2:** Bật server bằng lệnh:
    ```bash
    python app.py
    ```
*   **Bước 3:** Server sẽ hoạt động tại địa chỉ: `http://127.0.0.1:5000/`.
*   **Lưu ý:** Đáp án (mật khẩu đúng của tài khoản `minh`) sẽ được hệ thống random ngẫu nhiên sau mỗi lần chạy lệnh bật server và in trực tiếp ra màn hình terminal.
